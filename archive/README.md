# Original Yggdrasil 1.0 distribution

`Yggdrasil-1.0-original.zip` is Mark Tarver's original Yggdrasil 1.0
distribution as published on shenlanguage.org, preserved here for
posterity. This repository's initial commit was unpacked from it.

- **Retrieved from:** the Internet Archive Wayback Machine —
  <https://web.archive.org/web/20240430183437/https://www.shenlanguage.org/Download/Yggdrasil.zip>
  (capture of 2024-04-30; the original
  `https://www.shenlanguage.org/Download/Yggdrasil.zip` is no longer
  served)
- **SHA-256:** `3a44f766ac0a4268683bdc2873f9c92a23faed514bf1163f286f1d7260607e3a`
- **Contents:** `yggdrasil.shen` (the 1.0 tree-shaker, Warshall-based),
  `commonlisp.shen` (the Common Lisp backend), `Primitives/CL/` (per-file
  primitive implementations), and a `KLambda/` snapshot of the
  Shen 34.6-era kernel (including `backend.kl` and the per-file stdlib)
  that 1.0 shook against.

Dr. Tarver's accompanying paper, *Using Yggdrasil to Generate
Stand-alone Programs from Shen* (30/08/2023, Shen Group), is preserved
at the repository root as [`yggdrasil.pdf`](../yggdrasil.pdf).
